
# Terminal Portfolio

A modern, terminal-themed portfolio website built with React and Next.js.

## Project Structure

```
src/
├── _components/
│   └── Terminal.tsx          # Main terminal UI component
├── app/
│   ├── globals.css           # Global styles
│   ├── layout.tsx            # Root layout
│   └── page.tsx              # Home page
├── hooks/
│   └── useMobile.tsx         # Mobile detection hook
```

## Features

- **Terminal Interface**: Interactive terminal-style UI for a unique portfolio experience
- **Responsive Design**: Mobile-friendly layout with `useMobile` hook
- **Modern Stack**: Built with Next.js and React

## Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation

```bash
git clone https://github.com/Moukhtar-youssef/terminal_portfolio.git
cd terminal_portfoil
bun install
```

### Development

```bash
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
bun run build
bun start
```
